Session 2: Building An Interactive Frontend


States and Hooks

import { useState } from "react";

export default function Counter() {
  // Declare a state variable "count" and a function "setCount" to update it
  const [count, setCount] = useState(0);

  return (
    <div className="flex flex-col items-center p-4">
      <h1 className="text-2xl font-bold mb-2">Counter: {count}</h1>
      <button 
        onClick={() => setCount(count + 1)} 
        className="px-4 py-2 bg-blue-500 text-white rounded-lg"
      >
        Increment
      </button>
    </div>
  );


useState(0) initializes the state variable count with 0.
setCount is a function used to update count.
Clicking the button updates count by increasing it by 1.
